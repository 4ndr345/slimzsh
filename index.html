Slimzsh

A small, usable configuration for ZSH

Download as .zip Download as .tar.gz View on GitHub

How to install?

$ curl https://raw.githubusercontent.com/michal-kazmierczak/slimzsh/gh-pages/installer | bash

After all set default shell to ZSH by typing:

$ chsh -s /bin/zsh
